Detailed Description

Dit is de PresentationLogic class, deze class zorgt er voor dat menu's goed worden opgebouwd.

Hoe je een menu item toevoegd, dit doe je in Menu hier is een lijst met PresentationModels daar voeg je dan een nieuw item aan toe zoals:

new PresentationModel(naam, beschrijving, welke clearance, presentation naam, bool voor submenu of niet), Zorg er voor dat als je in een nieuwe presentation komt dat je dit doet: PresentationLogic.CurrentPresentation = "movie"; dan als je PresentationLogic.WriteMenu(Menu.presentationModels, true|false); aanroept worden alleen de menu items van movie getoont. Als je een nieuwe presentation aanmaakt zorg dan dat deze de interface IPresentation overneemt, die class moet dan Start() implementeren, in deze start schrijf je het menu, maar maar je ook een "state machine", dit kan heel simpel door gewoon een opties aan een switch/if statement te doen. (Tip: kijk naar de andere presentation Start() methods).
